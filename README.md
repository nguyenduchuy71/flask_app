## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install foobar.

```bash
pip install virtualenv

python3 -m venv env

source env/bin/activate

pip install --upgrade pip

pip install -r requirements.txt
```


## Run

```bash
python3 server.py
```
